# 🎵 AI Music Generation using LSTM

This project generates music using deep learning (LSTM) trained on MIDI datasets.

## 🔧 Technologies Used
- Python
- TensorFlow / Keras
- music21
- NumPy

## 🚀 Features
- Generates new music sequences
- Uses temperature-based sampling
- Produces multiple unique outputs when run multiple times

## 📂 Project Structure
- notebook/ → main implementation
- model/ → trained model
- outputs/ → generated MIDI files

## ▶️ How to Run

1. Download or clone this repository

2. Open the notebook in Kaggle, Jupyter Notebook, or Google Colab

3. Install required libraries:

   * music21
   * tensorflow
   * numpy

4. Attach the MIDI dataset from the Kaggle link below

5. Run the notebook cells sequentially from top to bottom

6. After training/loading the model, run the music generation cells

7. Generated `.mid` music files will be saved in the output folder

## ⚠️ Note

* This notebook already contains outputs for reference
* If running on Kaggle, attach the dataset before executing the notebook
* The pretrained model can be used directly without retraining
