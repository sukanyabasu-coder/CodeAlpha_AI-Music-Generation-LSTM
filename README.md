# 🎵 AI Music Generation using LSTM

This project generates music using deep learning (LSTM) trained on MIDI datasets.

## 🔧 Technologies Used
- Python
- TensorFlow / Keras
- music21
- NumPy

## 🚀 Features
- Generates new music sequences
- Uses temperature-based sampling
- Produces multiple unique outputs when run multiple times

## 📂 Project Structure
- notebook/ → main implementation
- model/ → trained model
- outputs/ → generated MIDI files

## ▶️ How to Run

1. Download the dataset from Kaggle (link below)
2. Open the notebook in Kaggle or Jupyter
3. Install required libraries:
   - music21
   - tensorflow
   - numpy
4. Run preprocessing and model loading cells
5. Run the generation cells to create music
6. Download the generated `.mid` files from the output folder
